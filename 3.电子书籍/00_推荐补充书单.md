# 推荐补充书单

这份书单用于记录适合继续补充的中文优质书籍。商业版权书只记录推荐，不直接下载；后续如果你有正版 PDF，可以再放进对应目录。

## 当前原则

- 优先中文：先保证能读懂、能吸收、能复盘。
- 优先和当前方向相关：Linux、C、内核、驱动、PCIe、存储、系统编程。
- 英文书暂不作为主线资料，只在以后需要查原版时再考虑。

## 技术类中文推荐

| 书名 | 方向 | 推荐理由 |
| --- | --- | --- |
| 《鸟哥的 Linux 私房菜 基础学习篇》 | Linux 基础 | 你已经有这本，适合反复查命令、目录、权限、服务 |
| 《鸟哥的 Linux 私房菜 服务器架设篇》 | Linux 服务 | 想补网络服务、系统管理时可以加 |
| 《Linux 命令行与 Shell 脚本编程大全》 | 命令与脚本 | Shell 语法、文本处理、脚本自动化更系统 |
| 《C Primer Plus 中文版》 | C 语言基础 | 适合把 C 语言语法、指针、数组、结构体补稳 |
| 《C 和指针》 | C 语言进阶 | 指针讲得细，对看驱动代码有帮助 |
| 《Linux/UNIX 系统编程手册》 | Linux 系统编程 | 文件、进程、信号、线程、IPC 都很硬核，适合长期查 |
| 《UNIX 环境高级编程》中文版 | 系统编程 | 经典系统编程书，适合配合实际代码看 |
| 《Linux 设备驱动开发详解》 | 驱动开发 | 中文驱动入门常见选择，适合字符设备、总线、模块 |
| 《深入理解 Linux 内核》中文版 | 内核机制 | 版本偏旧，但进程、内存、中断等概念适合打底 |
| 《PCI Express 体系结构导读》 | PCIe | 你已经有这本，适合继续作为 PCIe 主线资料 |
| 《大话存储》 | 存储系统 | 对磁盘、阵列、存储系统整体认识有帮助 |

## 阅读优先级

1. Linux 命令和 Shell：先看《鸟哥的 Linux 私房菜 基础学习篇》和现有基础笔记。
2. C 语言：优先《C Primer Plus 中文版》，再看《C 和指针》。
3. 系统编程：优先《Linux/UNIX 系统编程手册》或《UNIX 环境高级编程》中文版。
4. 驱动和内核：先《Linux 设备驱动开发详解》，再《深入理解 Linux 内核》中文版。
5. PCIe 和存储：继续看《PCI Express 体系结构导读》，补《大话存储》。
