# 期权策略学习笔记

整理日期：2026-06-30  
用途：理解期权卖方策略，不把“收权利金”误以为无风险理财。

> 原则：卖期权不是稳定理财，而是卖波动率、收时间价值，并承担义务。

## 四个基础动作

| 动作 | 英文 | 含义 |
|---|---|---|
| 买入开仓 | Buy to Open | 买入一张新期权，成为权利方 |
| 卖出平仓 | Sell to Close | 卖掉之前买入的期权，退出仓位 |
| 卖出开仓 | Sell to Open | 卖出一张新期权，成为义务方 |
| 买入平仓 | Buy to Close | 买回之前卖出的期权，解除义务 |

## Call 和 Put

| 类型 | 买方权利 | 卖方义务 |
|---|---|---|
| Call 看涨期权 | 有权按执行价买入股票 | 被行权时，按执行价卖出股票 |
| Put 看跌期权 | 有权按执行价卖出股票 | 被行权时，按执行价买入股票 |

标准美股期权通常 1 张等于 100 股。

```text
实际权利金 = 期权报价 x 100 x 合约张数
```

## 相对保守的卖方策略

### 现金担保 Put

适合：本来就愿意低价买入某只股票或 ETF。

例子：

```text
MSFT 当前 380
卖出 350 Put
收到权利金 5 x 100 = 500 美元
若跌破 350，可能被迫用 350 买入 100 股
实际成本约 350 - 5 = 345
```

风险：股价继续大跌，接货后浮亏。

### 备兑 Call

适合：已经持有 100 股正股，并愿意在更高价格卖出。

例子：

```text
持有 100 股 MSFT
卖出 420 Call
收到权利金
如果 MSFT 涨破 420，可能被迫 420 卖出股票
```

风险：股票下跌照样亏；股票大涨时收益被封顶。

## 危险策略

| 策略 | 为什么危险 |
|---|---|
| 裸卖 Call | 没有股票却卖 Call，理论亏损无限 |
| 保证金卖 Put | 没有足够现金接货，遇到大跌可能爆仓 |
| 财报前卖期权 | 波动不确定，一次可能亏掉很久权利金 |
| 卖垃圾股高权利金 | 权利金高通常对应真实高风险 |
| 一周到期近价期权 | Gamma 风险大，容易被快速打穿 |

## 卖方的真实收益特征

```text
高胜率 + 小利润 + 偶尔大亏
```

卖方赚的是：

1. 时间流逝。
2. 实际波动小于市场预期波动。
3. 标的没有朝不利方向大幅移动。

卖方亏的是：

1. 股价单边大跌。
2. 股价单边大涨。
3. 波动率突然升高。
4. 仓位过大导致无法扛波动。

## 我的期权使用边界

| 条件 | 是否允许 |
|---|---|
| 有现金，卖想买标的的 Put | 可以小仓学习 |
| 有 100 股，卖备兑 Call | 可以小仓学习 |
| 没股票裸卖 Call | 禁止 |
| 没现金卖 Put | 禁止 |
| 用消费贷做期权 | 禁止 |
| 财报前卖近价期权 | 禁止 |
| 总期权义务超过可承受接货金额 | 禁止 |

## 入门练习顺序

1. 先只看期权链，不交易。
2. 记录同一标的不同执行价、不同到期日的权利金。
3. 用纸面交易模拟 Sell to Open 和 Buy to Close。
4. 只研究 QQQ、SPY、MSFT、AAPL 这种高流动性标的。
5. 第一次真实交易只用最小仓位。
6. 每次交易前填写 `交易前检查清单.md`。
