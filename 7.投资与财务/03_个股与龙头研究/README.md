## 使用原则

- 七姐妹不是同一种股票：MSFT/AAPL/GOOGL/AMZN 更像平台型现金流，NVDA 更像 AI 算力周期，META 是高波动广告/AI 平台，TSLA 是高波动叙事资产。
- 不因为它们是龙头就默认一定很快涨回去；先看回撤深度、趋势、波动率和基本面有没有变化。
- 单股 2x、sell put、put spread 都必须先看仓位和现金，不把“龙头”当成杠杆理由。
