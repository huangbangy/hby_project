# 实盘跟踪更新流程

整理日期：2026-06-30  
用途：规定实盘看板多久更新、更新什么、更新后如何决策。

> 原则：市场每天变，但策略不用每天变。实盘体系要稳定、可重复、可复盘。

## 每周更新

频率：每周一次，建议周末。

| 步骤 | 操作 |
|---|---|
| 1 | 更新 `实盘持仓与目标仓位看板.md` 的当前价、市值、占比 |
| 2 | 更新现金余额和应急金月数 |
| 3 | 检查是否有单一标的超配 |
| 4 | 检查是否有标的接近触发价 |
| 5 | 写一句本周结论：加仓、观望、减仓、补现金 |

## 大跌时更新

触发：QQQ 单日跌 2% 以上，或重点个股单日跌 5% 以上。

| 步骤 | 操作 |
|---|---|
| 1 | 打开 `重点观察标的触发价.md` |
| 2 | 判断跌幅是否只是普通波动，还是触发计划买点 |
| 3 | 打开 `交易前检查清单.md` 填写买入理由 |
| 4 | 检查现金是否足够，不足则不加杠杆 |
| 5 | 若买入，记录到 `月度投资复盘.md` |

## 每月复盘

频率：每月最后一个交易日或月末周末。

| 步骤 | 操作 |
|---|---|
| 1 | 填写 `月度投资复盘.md` 的总资产、收益率、最大回撤 |
| 2 | 记录本月交易和是否符合规则 |
| 3 | 检查现金比例是否低于目标 |
| 4 | 检查 TQQQ/2x/期权是否超限 |
| 5 | 写出下月最重要的一条纪律 |

## 每季度更新

频率：每季度一次。

| 步骤 | 操作 |
|---|---|
| 1 | 更新重点标的历史高点和触发价 |
| 2 | 复查 QQQ/TQQQ 回撤策略是否仍适用 |
| 3 | 复查 MSFT、VST、COST、AVGO 等标的基本面 |
| 4 | 检查跨境资金、W-8BEN、出入金记录 |
| 5 | 根据收入和现金流调整目标仓位 |

## 决策流程

```text
先看现金流 -> 再看仓位 -> 再看触发价 -> 再填交易检查 -> 最后执行交易
```

任何一次交易，如果跳过 `交易前检查清单.md`，都视为情绪交易。

## 需要更新的文件顺序

1. `个人资产负债与现金流表.md`
2. `实盘持仓与目标仓位看板.md`
3. `重点观察标的触发价.md`
4. `交易前检查清单.md`
5. `月度投资复盘.md`

## 禁止事项

1. 不因为单日大跌就立刻满仓。
2. 不因为看到触发价就自动买入。
3. 不在现金低于 3 个月生活费时加杠杆。
4. 不在没有卖出计划时买入。
5. 不在未记录原因时做期权交易。
