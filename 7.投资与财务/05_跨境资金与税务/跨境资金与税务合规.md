## 税务关注点

| 场景 | 常见处理 |
|---|---|
| 美股资本利得 | 美国通常不对非美国税务居民征资本利得税；仍需关注本国税务居民申报义务 |
| 美股股息 | 通常会被美国预扣股息税，W-8BEN 信息很重要 |
| 港股交易 | 香港通常有股票印花税 |
| 港股资本利得 | 香港通常没有一般资本利得税 |
| CRS 信息交换 | 香港金融机构可能按税务居民身份交换账户信息 |
| 出金转账本身 | 通常不是纳税事件，但可能触发银行资金来源询问 |

## 禁止事项

1. 不用他人账户代收代付。
2. 不拆分交易来规避银行审核。
3. 不把贷款资金包装成生活费后流入证券账户。
4. 不借别人购汇额度。
5. 不把 Wise 当作绕开合规的工具。
6. 不忽略税务居民身份和年度记录保存。
