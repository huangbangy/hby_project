# 投资与财务

这里放投资相关数据分析和个人财务思考。

## 当前内容

- `投资策略总览_当前观察清单.md`：把当前价格、历史高点、回撤触发价、观察优先级、仓位框架和止盈规则放在一张总览里，作为本目录的优先入口。
- `个人资产负债与现金流表.md`：记录收入、支出、现金、投资仓位和负债，判断自己能承受多少波动。
- `实盘持仓与目标仓位看板.md`：记录真实持仓、成本、市值、目标仓位和风险暴露。
- `重点观察标的触发价.md`：集中记录 QQQ、MSFT、VST、COST、AVGO 等重点标的的回撤触发价。
- `实盘跟踪更新流程.md`：规定每周、每月、大跌时和每季度该怎么更新实盘体系。
- `仓位与风险控制规则.md`：给 QQQ、MSFT、TQQQ、单股、期权和现金设置仓位上限与禁止事项。
- `交易前检查清单.md`：每次买入、卖出、换仓、卖期权前填写，减少情绪交易。
- `期权策略学习笔记.md`：记录 Call/Put、Sell to Open、备兑 Call、现金担保 Put 和裸卖风险。
- `跨境资金与税务合规.md`：记录 Firstrade、香港汇丰、Wise、老虎证券、W-8BEN、股息税和出入金路径。
- `月度投资复盘.md`：每月复盘资产、收益、回撤、仓位和纪律执行情况。
- `往年数据分析.txt`：QQQ/TQQQ 等历史下跌与恢复数据分析。
- `QQQ_TQQQ回撤买入统计.md`：QQQ 从高点回撤不同幅度后买入 TQQQ，并持有到 QQQ 回前高的历史统计表和策略结论。
- `个股_2x杠杆回撤买入统计_AAPL_MSFT_AMZN.md`：AAPL/MSFT/AMZN 与对应 2x 单股杠杆 ETF 的回撤买入统计、年度触发频率和策略结论。
- `扩展龙头_2x杠杆回撤买入统计.md`：GOOGL/META/NVDA/TSLA/AVGO/AMD/NFLX/COST/JPM/BRK-B 的模拟 2x 深回撤修复统计和龙头筛选结论。

## 建议使用顺序

1. 先填 `个人资产负债与现金流表.md`，确认现金是否足够。
2. 再填 `实盘持仓与目标仓位看板.md`，确认当前真实仓位。
3. 看 `仓位与风险控制规则.md`，决定这次最多能买多少。
4. 用 `重点观察标的触发价.md` 判断是否真的到买点。
5. 每次交易前填 `交易前检查清单.md`。
6. 每月用 `月度投资复盘.md` 总结一次。
7. 按 `实盘跟踪更新流程.md` 做每周、每月和季度更新。
8. 需要研究策略时，再打开 QQQ/TQQQ、个股 2x、扩展龙头和期权笔记。
