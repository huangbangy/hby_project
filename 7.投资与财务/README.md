## 建议使用顺序

1. 先填 [个人资产负债与现金流表.md](01_个人财务与仓位/个人资产负债与现金流表.md)，确认现金和负债压力。
2. 再更新 [实盘持仓与目标仓位看板.md](01_个人财务与仓位/实盘持仓与目标仓位看板.md)，确认当前真实仓位。
3. 看 [仓位与风险控制规则.md](01_个人财务与仓位/仓位与风险控制规则.md)，决定这次最多能买多少。
4. 用 [重点观察标的触发价.md](01_个人财务与仓位/重点观察标的触发价.md) 判断是否真的到买点。
5. 每次交易前填 [交易前检查清单.md](00_入口与流程/交易前检查清单.md)。
6. 每月用 [月度投资复盘.md](00_入口与流程/月度投资复盘.md) 总结一次。
7. 按 [实盘跟踪更新流程.md](00_入口与流程/实盘跟踪更新流程.md) 做每周、每月和季度更新。
8. 需要研究策略时，再打开指数、个股、期权或跨境资金对应文件夹。

## 使用原则

- `00_入口与流程` 是每天最多打开的地方。
- `01_个人财务与仓位` 是交易前必须确认的地方。
- `02/03/04` 是研究区，不是每天反复盯盘的地方。
- `99_原始记录` 保留原始想法，尽量不直接改；新的结论沉淀到前面的分类文档里。
