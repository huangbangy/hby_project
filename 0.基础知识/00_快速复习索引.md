## 高频命令

| 场景 | 命令 |
| --- | --- |
| 看内核版本 | `uname -r` |
| 看 CPU 型号 | `cat /proc/cpuinfo | grep 'model name'` |
| 看内存 | `free -h`、`dmidecode -t memory` |
| 看磁盘占用 | `df -h`、`du -h --max-depth=1` |
| 追日志 | `tail -f /var/log/messages` |
| 查 PCI 设备 | `lspci -tv`、`lspci -vvv -s <bus:dev.func>` |
| 查 I2C 总线 | `i2cdetect -l`、`ls /sys/bus/i2c/devices/` |
| 加载模块 | `modprobe <module>` |
| 后台运行脚本 | `nohup ./test.sh &` |
