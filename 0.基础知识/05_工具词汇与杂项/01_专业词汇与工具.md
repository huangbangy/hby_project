# 专业词汇与工具

这里放硬件、存储、互连、系统调试里经常遇到的词汇，适合面试前快速过一遍。

## 硬件与平台

| 词汇 | 全称/中文 | 解释 | 记忆点 |
| --- | --- | --- | --- |
| Flash | 闪存 | 断电后仍能保存数据的非易失性存储器 | BIOS、固件、SSD 都可能用到 |
| BMC | Baseboard Management Controller，基板管理控制器 | 服务器中的独立管理芯片/处理器，负责监控和管理硬件状态 | 可管理风扇、电源、温度、传感器，常通过 I2C 与设备通信 |
| 1U / 2U | Rack Unit，机架高度单位 | 1U = 1.75 英寸，2U = 3.5 英寸 | 用来描述服务器高度 |
| LPC | Low Pin Count Bus | PC 兼容机中连接低带宽或传统设备的总线 | 常见于老式外设、Super I/O、TPM 等场景 |
| `/proc/ioports` | I/O 端口分配表 | 显示系统 I/O 端口地址分配 | 调试 ISA、PCI、LPC 设备时有用 |

## 存储与接口

| 词汇 | 解释 | 记忆点 |
| --- | --- | --- |
| U.2 | 常见企业级 SSD 物理接口规格 | 可承载 PCIe/NVMe，也可能与具体背板设计有关 |
| M.2 | 小型 SSD/无线模块等常见接口规格 | 常见于笔记本、工控机、开发板 |
| NVMe | 基于 PCIe 的高性能存储协议 | 不是物理口，是协议 |
| SATA | 存储接口和传输协议 | 常见普通 SSD/HDD |
| iSCSI | 把 SCSI 命令封装在 TCP/IP 上传输的协议 | 让远端存储像块设备一样被访问 |

## 高速互连与内存

| 词汇 | 全称 | 解释 |
| --- | --- | --- |
| CXL | Compute Express Link | 面向 CPU、内存、加速器、存储设备的高速低延迟互连标准 |
| RDMA | Remote Direct Memory Access | 远程直接内存访问，数据可在网卡之间直接搬运，减少 CPU 参与 |
| 主频 | Clock Frequency | 常见理解：主频 = 外频 x 倍频 |

## 工具与版本管理

| 工具 | 用途 | 备注 |
| --- | --- | --- |

## 证书方向速记

如果未来从事程序员或计算机技术相关岗位，可以优先考虑：

- 中级：软件设计师。
- 高级：系统架构设计师。

如果不想走程序员方向，更偏项目管理或信息化管理，可以考虑：

- 中级：系统集成项目管理工程师。
- 高级：信息系统项目管理师。
