# 每日学习总结

这里保留阶段性学习收获，并把原来的散点内容整理成时间线和知识点。

## 2024-06-19

### I2C 基础

- I2C 协议包含主设备和从设备。
- 每个从设备有唯一地址。
- 总线主要由 SCL、SDA 和上拉电阻构成。
- 开始信号：SCL 为高电平时，SDA 从高到低。
- 结束信号：SCL 为高电平时，SDA 从低到高。

### 常见通信协议

| 协议 | 常见场景 |
| --- | --- |
| I2C | 低速传感器、EEPROM、电源芯片 |
| SPI | Flash、ADC、显示屏等较高速片间通信 |
| UART | 串口调试、模块通信 |
| CAN | 汽车、工业控制 |
| USB | 外设连接 |
| Bluetooth / Wi-Fi | 无线通信 |

## 2024-06-27

### 磁盘与模块

```bash
dd if=/dev/zero of=/dev/sdb bs=1K count=1
gdisk /dev/sdb
dmesg
modinfo /lib/modules/3.10.0/kernel/drivers/net/ethernet/intel/i40e/i40e.ko
```

要点：

- `dd if=/dev/zero of=/dev/sdb bs=1K count=1` 会向 `/dev/sdb` 写入 1KB 的零数据，可能破坏磁盘数据。
- `gdisk` 用于 GPT 分区表操作。
- `dmesg` 查看内核日志。
- `modinfo` 查看内核模块信息。

## 2024-06-28

### PCIe 查询

```bash
lspci -tv
lspci -nn
lspci -s 0000:0c:00.0 -vvv
```

| 命令 | 用途 |
| --- | --- |
| `lspci -tv` | 查看 PCI 设备树 |
| `lspci -nn` | 查看设备名称、厂商 ID、设备 ID |
| `lspci -s <BDF> -vvv` | 查看指定设备详细信息，包括链路能力和速率 |

### 文本处理组合

```bash
cat test.txt | grep -i "hby" | awk '{print $1}' | sed -n '1p'
```

含义：忽略大小写查找 `hby`，取第 1 列，再打印第 1 行。

### 网络设备信息

```bash
lshw -C network
```

用于查看网卡型号、制造商、驱动、连接状态、速率等信息。

### 磁盘 SMART 信息

```bash
smartctl -i /dev/sda
```

可查看磁盘制造商、型号、固件版本、序列号、容量和扇区大小等信息。

### systemctl 常见用法

| 命令 | 用途 |
| --- | --- |
| `systemctl start service-name` | 启动服务 |
| `systemctl stop service-name` | 停止服务 |
| `systemctl restart service-name` | 重启服务 |
| `systemctl status service-name` | 查看服务状态 |
| `systemctl enable service-name` | 设置开机自启 |
| `systemctl disable service-name` | 取消开机自启 |
| `systemctl is-enabled service-name` | 查看是否开机自启 |
| `systemctl is-active service-name` | 查看是否正在运行 |
| `systemctl list-units --type=service` | 列出服务单元 |
| `journalctl` | 查看 systemd 日志 |

### 动态库路径

```bash
export LD_LIBRARY_PATH=/path/to/library
```

`LD_LIBRARY_PATH` 用于告诉系统运行程序时去哪些目录查找共享库。临时调试很方便，但正式部署中更推荐使用系统库路径、`ldconfig` 或 rpath 等方式。
