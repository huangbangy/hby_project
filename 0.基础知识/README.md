# 基础知识

这里存放适合反复查阅的基础技术笔记，主要覆盖 Linux、Shell、C 语言、内核调试、PCIe、I2C 和常见工具词汇。

## 入口

- [00_快速复习索引.md](00_快速复习索引.md)：按问题和场景快速定位资料。
- [01_基础命令与脚本](01_基础命令与脚本)：Linux 常用命令、Shell 脚本写法。
- [02_C语言与基础概念](02_C语言与基础概念)：C 语言、编译链接、内存、信号量、锁。
- [03_Linux内核与调试](03_Linux内核与调试)：GDB、内核编译、模块、日志、内核调试。
- [04_PCIe与硬件](04_PCIe与硬件)：PCIe、BAR、MSI、I2C、硬件时序。
- [05_工具词汇与杂项](05_工具词汇与杂项)：专业词汇、工具、学习总结和暂未细分的技术记录。
- [99_原始备份](99_原始备份)：原始未拆分内容，便于回溯。

## 使用建议

1. 新知识先随手写到根目录的 [0_知识大杂烩.md](../0_知识大杂烩.md)。
2. 如果某条内容已经能归类，再移动到本目录对应专题。
3. 面试复盘、岗位 JD、简历素材统一放到 [1.面试资料](../1.面试资料)。
4. 不确定是否有用的零散记录，可以先留在 `05_工具词汇与杂项/03_技术随手记录_2024.md`，后面再拆。

## 整理规则

- 能回答“是什么”的内容，放到概念类文件。
- 能回答“怎么查、怎么做”的内容，放到命令或操作类文件。
- 一次事故、一次定位、一次工作记录，先放到随手记录，后续再提炼成知识点。
- 每个文件尽量保留“速查表 + 示例 + 注意点”的结构，方便以后复习。
