# 基础知识

这里存放基础类知识笔记，主要由原来的 `0_知识大杂烩` 拆分而来。

## 目录说明

- `01_基础命令与脚本`：Linux 常用命令、Shell 脚本写法。
- `02_C语言与基础概念`：C 语言、指针、内存、信号量等基础内容。
- `03_Linux内核与调试`：GDB、内核编译、内核调试、日志分析等。
- `04_PCIe与硬件`：PCIe、BAR、MSI、I2C、硬件时序等。
- `05_工具词汇与杂项`：专业词汇、工具、每日总结和暂未细分的技术记录。
- `99_原始备份`：原始未拆分内容，便于回溯。

岗位 JD 类内容统一放在 `1.面试资料/02_岗位职责与JD`。
