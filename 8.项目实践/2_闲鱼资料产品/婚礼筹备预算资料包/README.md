# 婚礼筹备预算资料包

面向备婚新人，提供预算、宾客、供应商沟通、婚礼前时间线和婚礼当天流程清单。

## 文件说明

- `婚礼筹备预算手册-正文.md`：主文档正文。
- `婚礼流程表与清单.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/wedding-planning-budget-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不提供婚庆执行、主持、摄影或法律服务。
- 不承诺婚礼效果或费用结果。
- 仅作婚礼筹备记录参考。
