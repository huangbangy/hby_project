# 租房验房清单

## 看房前准备

- [ ] 确认预算上限
- [ ] 确认通勤时间
- [ ] 确认房源身份：房东/中介/转租
- [ ] 准备拍照或录视频
- [ ] 列出必须条件和可妥协条件

## 看房现场

- [ ] 门锁是否正常
- [ ] 窗户是否能关紧
- [ ] 墙面是否发霉
- [ ] 天花板是否渗水
- [ ] 卫生间是否反味
- [ ] 水龙头水压是否正常
- [ ] 马桶冲水是否正常
- [ ] 地漏排水是否顺畅
- [ ] 插座是否够用
- [ ] 空调是否制冷/制热
- [ ] 冰箱是否正常
- [ ] 洗衣机是否正常
- [ ] 床、柜子、桌椅是否损坏
- [ ] 楼道和电梯是否正常
- [ ] 晚上是否吵闹

## 合同检查

- [ ] 双方身份信息清楚
- [ ] 房屋地址准确
- [ ] 租期起止时间明确
- [ ] 租金金额明确
- [ ] 押金金额明确
- [ ] 付款周期明确
- [ ] 水电燃气物业网费承担方式明确
- [ ] 家具家电清单明确
- [ ] 维修责任明确
- [ ] 提前退租规则明确
- [ ] 转租规则明确
- [ ] 押金退还时间明确

## 入住交接

```text
房屋地址：

```

## 退租检查

- [ ] 提前按合同通知
- [ ] 拍退租前视频
- [ ] 清洁房间
- [ ] 恢复家具原位
- [ ] 结清水电燃气网费
- [ ] 归还钥匙门禁
- [ ] 确认押金退还金额
- [ ] 确认押金退还时间
