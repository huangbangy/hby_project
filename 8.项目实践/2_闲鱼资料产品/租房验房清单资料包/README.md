# 租房验房清单资料包

面向第一次租房、换租和异地工作人群，用于看房、验房、合同、交接、搬家和退租押金检查。

## 文件说明

- `租房验房手册-正文.md`：主文档正文。
- `租房验房清单.md`：可复制清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/rental-checklist-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 只提供生活经验和检查清单，不构成法律意见。
- 不代签合同，不处理纠纷，不承诺追回押金。
- 涉及合同争议时，应咨询专业人士或当地主管部门。
