# 小店客服售后话术资料包

面向个人闲置卖家、手作小店和副业卖家，提供买家咨询、发货说明、售后沟通、高频问题知识库和复盘表。

## 文件说明

- `小店客服售后话术手册-正文.md`：主文档正文。
- `客服话术表与清单.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/small-shop-customer-service-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不代运营店铺或发布商品。
- 不提供虚假互动或违规引导。
- 不承诺成交、好评或售后处理结果。
