# 闲鱼资料产品

这里放适合做成原创 PDF / 模板包的小型资料产品。每个项目尽量包含：

- 正文手册
- 模板或清单
- 闲鱼上架文案
- 交付版 PDF
- 上架预览图片

## 当前项目

- `AI办公效率资料包`：AI 办公、写作、求职、闲鱼卖家提示词模板。
- `求职简历面试资料包`：简历优化、岗位匹配、面试问答和求职沟通模板。
- `大学生学习规划资料包`：大学四年规划、自学计划、考证实习和复盘模板。
- `租房验房清单资料包`：看房、验房、合同、交接、搬家和押金清单。

## 工具

- `tools/build-product-assets.mjs`：根据每个项目的 `product.json` 生成 PDF 和上架图片。

示例：

```bash
node tools/build-product-assets.mjs "8.项目实践/2_闲鱼资料产品/求职简历面试资料包"
```
