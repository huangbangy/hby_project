# 闲鱼资料产品

这里放适合做成原创 PDF / 模板包的小型资料产品。每个项目尽量包含：

- 正文手册
- 模板或清单
- 闲鱼上架文案
- 交付版 PDF
- 上架预览图片

## 当前项目

- `AI办公效率资料包`：AI 办公、写作、求职、闲鱼卖家提示词模板。
- `求职简历面试资料包`：简历优化、岗位匹配、面试问答和求职沟通模板。
- `大学生学习规划资料包`：大学四年规划、自学计划、考证实习和复盘模板。
- `租房验房清单资料包`：看房、验房、合同、交接、搬家和押金清单。
- `职场汇报PPT资料包`：周报月报、述职复盘、项目汇报和演讲稿结构模板。
- `个人记账预算资料包`：月度预算、消费复盘、存钱计划和家庭账本清单。
- `备考规划复盘资料包`：考研考公考证通用的学习计划、错题复盘和进度表。
- `内容选题发布资料包`：账号定位、选题库、标题模板、发布计划和复盘表。
- `装修预算验收资料包`：装修预算、材料清单、施工沟通和验收记录模板。
- `旅行出行清单资料包`：行李清单、预算规划、出行日程和返程复盘模板。
- `婚礼筹备预算资料包`：婚礼预算、宾客名单、流程安排和物品清单模板。
- `新员工试用期资料包`：入职计划、工作记录、汇报复盘和沟通模板。
- `二手车看车清单资料包`：看车记录、试驾检查、费用估算和交易资料核对模板。
- `搬家打包交接资料包`：搬家计划、打包标签、物品清单、租房交接和入住复盘模板。
- `家庭收纳整理资料包`：空间盘点、断舍离清单、物品归位和日常维护模板。
- `宠物新手照护资料包`：接宠准备、用品清单、喂养记录、日常观察和就医资料模板。
- `小店客服售后话术资料包`：买家问答、发货说明、售后沟通和复盘模板。

## 上架执行

- `闲鱼上架执行清单.md`：标题、图片顺序、详情页结构、客服第一句和避坑表达。

## 工具

- `tools/build-product-assets.mjs`：根据每个项目的 `product.json` 生成 PDF 和上架图片。

示例：

```bash
node tools/build-product-assets.mjs "8.项目实践/2_闲鱼资料产品/求职简历面试资料包"
```
