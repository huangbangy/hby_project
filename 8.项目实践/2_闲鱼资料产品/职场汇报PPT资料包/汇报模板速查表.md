# 汇报模板速查表

用途：复制后替换括号内容即可使用。适合周报、月报、述职、项目复盘和 PPT 初稿。

## 周报模板

```text
本周目标：【目标】
已完成：
1. 【事项】——结果：【结果】
2. 【事项】——进度：【进度】

遇到问题：【问题】
下周计划：【动作】
需要协同：【对象+事项】
```

## 月报模板

```text
本月重点：【重点】
关键成果：【成果】
数据变化：【数据】
问题与原因：【问题+原因】
下月计划：【计划】
需要支持：【支持】
```

## 述职模板

```text
我的职责：【职责】
今年重点：【重点】
关键动作：【动作】
结果反馈：【结果】
经验沉淀：【经验】
下一阶段计划：【计划】
```

## 项目复盘模板

```text
项目背景：【背景】
目标：【目标】
关键动作：【动作】
结果：【结果】
问题：【问题】
原因：【原因】
改进：【下次怎么做】
```

## PPT 页面标题模板

```text
本页结论：【一句话结论】
支撑信息：【数据/案例/过程】
下一步：【动作/判断/建议】
```

## 演讲稿模板

```text
这一页主要说明【主题】。
可以看到【关键事实】。
所以我的结论是【结论】，接下来会【动作】。
```

## 自检模板

```text
请检查这份汇报：
1. 是否有一句话结论？
2. 每页标题是否清楚？
3. 数据是否真实可信？
4. 问题是否有原因分析？
5. 下一步动作是否具体？
```
