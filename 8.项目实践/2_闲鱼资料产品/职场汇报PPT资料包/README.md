# 职场汇报PPT资料包

面向需要写周报、月报、述职、项目复盘和 PPT 大纲的职场人，提供可复制的汇报结构和自检清单。

## 文件说明

- `职场汇报PPT手册-正文.md`：主文档正文。
- `汇报模板速查表.md`：可复制模板。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/workplace-report-ppt-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不代写公司正式材料。
- 不编造业绩、数据或项目经历。
- 不承诺晋升、考核或汇报结果。
