# 装修预算验收资料包

面向准备装修、旧房翻新、自装或半包用户，提供预算拆分、材料记录、施工沟通、增项变更和验收问题清单。

## 文件说明

- `装修预算验收手册-正文.md`：主文档正文。
- `预算验收表与清单.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/renovation-budget-check-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不提供施工、设计、法律或质量鉴定建议。
- 不承诺装修效果或节省金额。
- 仅作家庭装修资料整理参考。
