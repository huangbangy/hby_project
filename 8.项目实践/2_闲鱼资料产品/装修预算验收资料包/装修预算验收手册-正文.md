## 1. 装修前先确认需求边界

装修最容易失控的地方，不是某一项突然变贵，而是前期没有把需求边界写清楚。

开工前建议先确认：

- 哪些空间必须改，哪些只是想优化。
- 哪些项目是刚需，哪些可以以后再做。
- 家里常住人数、收纳需求、宠物和老人小孩需求。
- 预算上限和预留机动金。
- 沟通人、付款人和现场确认人是谁。

需求越清楚，后面临时变更就越少。

## 2. 预算怎么拆到每个空间

不要只写一个总预算。建议按空间和工种拆开：

| 空间 | 项目 | 预算 | 已支付 | 备注 |
|---|---|---:|---:|---|
| 客厅 | 地面/墙面/灯具 |  |  |  |
| 厨房 | 橱柜/电器/水电 |  |  |  |
| 卫生间 | 防水/洁具/吊顶 |  |  |  |
| 卧室 | 衣柜/地板/窗帘 |  |  |  |

建议把预算分成三类：

- 必须支出：基础施工、必要主材、基础电器。
- 可调整支出：灯具、窗帘、软装、部分家具。
- 预留支出：增项、维修、补货、运输安装。

预留金建议单独写出来，不要混在总预算里。

## 3. 主材辅材记录方法

材料记录至少包含品牌、型号、规格、数量、单价、下单时间、预计到货时间。

```text
材料名称：【名称】
使用位置：【空间】
品牌型号：【信息】
规格数量：【数量】
单价总价：【金额】
购买渠道：【门店/平台】
到货时间：【日期】
安装责任：【人员】
```

同一类材料不要只拍一张图片，最好把标签、订单、发票或收据一起留存。

## 4. 施工节点沟通表

每个节点沟通后，建议留下文字记录。

| 日期 | 节点 | 沟通事项 | 确认结果 | 负责人 |
|---|---|---|---|---|
|  | 水电定位 | 插座数量和位置 |  |  |
|  | 泥工 | 瓷砖铺贴方式 |  |  |
|  | 木工 | 柜体尺寸 |  |  |

沟通记录不是为了争执，而是让双方对“已经确认过什么”有共同依据。

## 5. 增项和变更记录

装修中出现变更很正常，关键是要记录清楚。

变更记录建议包含：

1. 变更原因。
2. 新增或减少的项目。
3. 金额变化。
4. 工期变化。
5. 谁提出、谁确认。
6. 是否已经付款。

```text
变更事项：【事项】
原方案：【内容】
新方案：【内容】
金额变化：【增加/减少金额】
工期影响：【天数】
确认人：【姓名】
确认日期：【日期】
```

## 6. 水电泥木油验收要点

验收时可以按节点记录问题：

- 水电：点位是否和需求一致，开关插座是否遗漏。
- 泥工：瓷砖空鼓、坡度、缝隙、阳角处理。
- 木工：尺寸、收口、五金安装、柜门开合。
- 油工：墙面平整度、阴阳角、色差、裂纹。
- 安装：灯具、洁具、门窗、五金是否可正常使用。

发现问题不要只口头说，建议写成问题清单并约定复查时间。

## 7. 收尾补漏清单

完工前容易遗漏的小项：

- 开关面板是否齐全。
- 地漏、龙头、马桶排水是否顺畅。
- 门窗开合是否顺畅。
- 柜门、抽屉、五金是否松动。
- 墙面是否有磕碰、污染或修补痕迹。
- 施工垃圾是否清理。
- 质保、票据、说明书是否归档。

收尾阶段最好逐项勾选，不要凭感觉说“差不多”。

## 8. 入住前复盘归档

入住前可以把资料分成四类保存：

1. 合同和报价。
2. 订单、收据和付款记录。
3. 材料型号和安装说明。
4. 验收问题和整改记录。

复盘问题：

- 哪些预算超了，原因是什么？
- 哪些需求一开始没有想清楚？
- 哪些材料或施工节点以后需要注意？
- 还有哪些售后事项需要追踪？

装修资料整理得越清楚，后续维修、补货和售后沟通就越省力。
