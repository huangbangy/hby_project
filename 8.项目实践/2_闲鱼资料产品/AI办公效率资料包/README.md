# AI 办公效率资料包

这是一个用于闲鱼资料产品测试的原创资料包。

## 文件说明

- `AI办公效率手册-正文.md`：主文档正文。
- `AI提示词速查表.md`：可复制提示词模板。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `上架图片/`：闲鱼商品图预览，包含主图、目录、模板预览、适合人群和交付说明。
- `dist/ai-office-manual.html`：生成后的 HTML 预览。
- `dist/ai-office-manual.pdf`：生成后的 PDF。
- `generate-listing-images.mjs`：重新生成上架图片的脚本。

## 上架图片顺序

1. `01-主图-AI办公效率手册.png`
2. `02-内容目录预览.png`
3. `03-提示词模板预览.png`
4. `04-适合人群与使用场景.png`
5. `05-交付说明与边界.png`

如需重新生成图片，在当前目录执行：

```bash
node generate-listing-images.mjs
```
