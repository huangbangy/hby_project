# 内容选题发布资料包

面向想做内容但缺少选题、标题、发布计划和复盘结构的新手创作者，提供定位表、选题库、发布计划和复盘模板。

## 文件说明

- `内容选题发布手册-正文.md`：主文档正文。
- `内容选题模板表.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/content-planning-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不代运营账号，不发布内容。
- 不提供虚假互动。
- 不承诺涨粉、成交或具体效果。
