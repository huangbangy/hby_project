# 宠物新手照护资料包

面向准备养宠、刚接宠和家庭共同照护场景，提供接宠准备、用品清单、喂养观察、异常记录和就医资料归档模板。

## 文件说明

- `宠物新手照护手册-正文.md`：主文档正文。
- `宠物记录表与清单.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/new-pet-care-record-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不提供诊疗、用药或训练保证。
- 不替代兽医检查和专业建议。
- 仅作宠物日常记录和新手准备参考。
