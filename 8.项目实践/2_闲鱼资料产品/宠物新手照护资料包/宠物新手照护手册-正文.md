## 1. 接宠前先准备什么

宠物刚到家，新手最容易手忙脚乱。不是因为不爱它，而是每件小事都像第一次。

接宠前建议先准备：

- 安静、安全的适应空间。
- 基础食物和饮水工具。
- 睡觉或休息区域。
- 清洁用品。
- 外出或就医用的包、牵引或航空箱。
- 最近的宠物医院联系方式。

不用第一天买齐所有东西，先保证安全、饮食、清洁和观察。

## 2. 到家第一周观察重点

第一周重点记录：

- 食欲和饮水。
- 精神状态。
- 排便排尿。
- 是否频繁躲藏、叫唤或焦躁。
- 是否出现持续异常表现。

```text
日期：【日期】
食欲：【正常/偏少/拒食】
饮水：【正常/偏少】
排便：【正常/异常】
精神：【活跃/安静/异常】
备注：【观察】
```

如果出现持续异常，建议及时咨询专业兽医。

## 3. 用品清单怎么分优先级

用品分三类：

| 优先级 | 用品 | 说明 |
|---|---|---|
| 必须准备 | 食盆、水盆、食物、清洁用品 | 到家当天就要用 |
| 尽快准备 | 外出包、指甲剪、梳子 | 日常护理会用到 |
| 以后再买 | 玩具、装饰、额外收纳 | 观察习惯后再买 |

新手不要被长清单吓到。先买会马上用到的东西。

## 4. 喂养和饮水记录

喂养记录可以帮助发现变化：

| 日期 | 食物 | 喂食量 | 饮水 | 状态 | 备注 |
|---|---|---:|---|---|---|
|  |  |  |  |  |  |

不要频繁更换食物。如果确实需要调整，建议按宠物实际情况循序渐进，并咨询专业建议。

## 5. 清洁护理日程表

常见日程：

- 每天：饮水、食盆、活动区域检查。
- 每周：清洁休息区域、梳毛、检查用品。
- 每月：整理记录、检查消耗品、归档票据。

不同宠物差异很大，日程只做记录参考。

## 6. 异常情况记录方式

遇到异常不要只写“感觉不对”，尽量记录事实：

```text
时间：【时间】
表现：【具体表现】
持续多久：【时长】
是否进食饮水：【情况】
是否拍照录像：【是/否】
已咨询或就医：【记录】
```

清楚的记录能帮助你和专业人士沟通。

## 7. 就医资料归档

建议归档：

- 宠物基础信息。
- 就医日期和医院。
- 检查项目和票据。
- 医生给出的建议。
- 后续观察事项。

健康相关决定请以专业兽医意见为准，资料包只帮助你把信息记录清楚。

## 8. 日常陪伴和复盘

每周可以复盘：

- 哪些用品真正有用？
- 哪些记录还没坚持？
- 宠物最放松的时间和位置？
- 家人之间分工是否清楚？
- 下周需要补充什么？

养宠不是把清单填满，而是慢慢建立稳定、安全、有爱的生活节奏。
