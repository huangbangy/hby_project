# 二手车看车清单资料包

面向二手车看车、试驾、多车对比和交易资料核对场景，提供信息记录、外观内饰检查、试驾记录和费用估算模板。

## 文件说明

- `二手车看车手册-正文.md`：主文档正文。
- `看车试驾表与清单.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/used-car-checklist-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不提供车辆鉴定、金融、保险或法律建议。
- 不承诺车况、价格或成交结果。
- 仅作个人看车记录参考。
