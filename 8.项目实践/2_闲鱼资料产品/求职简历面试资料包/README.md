# 求职简历面试资料包

面向应届生、转行者和职场新人，用于整理简历、匹配岗位、准备面试和规范求职沟通。

## 文件说明

- `求职简历面试手册-正文.md`：主文档正文。
- `求职模板速查表.md`：可复制模板。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/job-search-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不编造学历、证书、工作经历、项目经历。
- 不承诺入职、内推、保过或面试结果。
- 只提供简历表达、结构梳理和面试准备模板。
