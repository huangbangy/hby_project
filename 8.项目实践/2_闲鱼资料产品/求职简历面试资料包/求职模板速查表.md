# 求职模板速查表

## 简历经历优化模板

```text
请帮我优化下面这段简历经历。
目标岗位：【岗位名称】
要求：
1. 使用动作 + 方法 + 结果的表达。
2. 不编造事实。
3. 缺少数据的地方用【可补充数据】标注。

原始经历：
【粘贴内容】
```

## STAR 经历拆解模板

```text
请把下面经历拆成 STAR 结构。
S 背景：
T 任务：
A 行动：
R 结果：
并帮我提炼适合写进简历的一句话。

经历：
【粘贴内容】
```

## JD 匹配模板

```text
请分析这个岗位 JD。
输出：
1. 必备条件
2. 核心能力
3. 加分项
4. 我的简历应该突出什么
5. 我缺少哪些能力

岗位 JD：
【粘贴 JD】
```

## 自我介绍模板

```text
面试官您好，我叫【姓名】，毕业于【学校/专业】。
我应聘的是【岗位】。我和岗位比较匹配的地方有【数量】点：
第一是【能力】，对应经历是【经历】；
第二是【能力】，对应经历是【经历】。
我希望在这个岗位上继续积累【方向】，也能为团队提供【价值】。
```

## 高频面试题清单

| 问题 | 准备重点 |
|---|---|
| 请做一下自我介绍 | 1 分钟，突出岗位匹配 |
| 为什么选择我们公司 | 公司业务 + 岗位匹配 |
| 为什么离职/转行 | 真实、克制、向前看 |
| 你的优势是什么 | 用经历证明 |
| 你的缺点是什么 | 可改善，不致命 |
| 项目中遇到最大困难 | 用 STAR 讲清楚 |
| 你有什么想问的 | 问岗位目标和团队期待 |

## 求职沟通模板

### 初次打招呼

```text
您好，我对贵公司的【岗位】很感兴趣。我有【相关经历/技能】，简历已发送，方便的话希望您帮忙看一下是否匹配，谢谢。
```

### 跟进进度

```text
您好，想跟进一下【岗位】的招聘进展。如果暂时不匹配，也希望能收到反馈，方便我继续调整，谢谢。
```

### 面试感谢

```text
您好，感谢今天的面试交流。通过沟通我对岗位和团队有了更清楚的了解，也更确认自己对这个方向感兴趣。期待后续反馈，谢谢。
```
