# 家庭收纳整理资料包

面向出租屋、小家庭、宿舍和换季整理场景，提供空间盘点、物品分类、断舍离判断、日常维护和家庭分工清单。

## 文件说明

- `家庭收纳整理手册-正文.md`：主文档正文。
- `收纳整理表与清单.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/home-declutter-organizing-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不提供上门收纳整理服务。
- 不承诺空间改造效果。
- 仅作家庭收纳和生活整理参考。
