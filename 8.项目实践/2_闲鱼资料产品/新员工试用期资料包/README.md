# 新员工试用期资料包

面向新员工、转行换岗和试用期自我管理场景，提供 30-60-90 天计划、工作记录、周报复盘和转正前自查清单。

## 文件说明

- `新员工试用期手册-正文.md`：主文档正文。
- `入职计划表与清单.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/new-employee-probation-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不提供劳动法律意见或职业中介服务。
- 不承诺转正、涨薪或晋升结果。
- 仅作职场工作记录和自我管理参考。
