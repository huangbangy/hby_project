# 大学生学习规划资料包

面向大学生，用于整理大学四年规划、自学计划、考证实习安排、简历素材和每周复盘。

## 文件说明

- `大学生学习规划手册-正文.md`：主文档正文。
- `学习规划表与清单.md`：可复制模板。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/student-study-plan-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不代写作业、论文、课程报告。
- 不售卖考试真题、押题或盗版资料。
- 不承诺考证、保研、上岸或录取结果。
