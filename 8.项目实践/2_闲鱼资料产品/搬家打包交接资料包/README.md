# 搬家打包交接资料包

面向租房换房、毕业搬家、家庭搬家和跨城搬家场景，提供搬家计划、纸箱编号、随身包、旧房交接和新家入住检查清单。

## 文件说明

- `搬家打包交接手册-正文.md`：主文档正文。
- `搬家打包表与清单.md`：可复制表格和清单。
- `闲鱼上架文案.md`：标题、详情页和客服回复。
- `product.json`：生成 PDF 和上架图的配置。
- `dist/moving-packing-handover-pack.pdf`：交付版 PDF。
- `上架图片/`：闲鱼商品图。

## 合规边界

- 不提供搬家服务、赔付或法律建议。
- 不承诺避免遗漏或损坏。
- 仅作搬家准备和资料记录参考。
