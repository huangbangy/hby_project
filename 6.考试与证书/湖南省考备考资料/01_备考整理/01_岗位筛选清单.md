# 岗位筛选清单

## 第一轮：硬条件排除

逐项排除不符合的岗位：

- 学历是否满足 `最低学历要求`
- 学位是否满足 `学位要求`
- 专业名称是否匹配 `专业要求`
- 是否满足 `报考人员身份要求`
- 是否满足 `基层工作年限要求`
- 是否满足 `最高年龄要求`
- 是否满足 `户籍要求`
- 是否满足 `职业资格证书等要求`
- 是否接受 `性别要求`
- 是否接受 `最低服务年限要求`
- 公安、执法类岗位是否接受体测、体检、政审要求

专业不确定时，先查 `02_湖南省2026年考试录用公务员专业指导目录.xlsx`。如果专业未列入目录，不要自己脑补能报，优先咨询招录单位。

## 第二轮：备考成本排序

保留能报岗位后，按考试科目排序：

| 类型 | 备考成本 | 说明 |
| --- | --- | --- |
| 行测 + 申论 | 低 | 主线备考即可 |
| 行测 + 申论行政执法卷 | 中 | 申论材料和答题角度要偏执法治理 |
| 加法律专业知识 | 高 | 适合法学背景或已有法考基础 |
| 加财经专业知识 | 高 | 适合财会、审计、金融背景 |
| 加公安专业科目 | 高 | 还要考虑体测、体检、政审 |

## 第三轮：胜算排序

给每个岗位打 1 至 5 分：

| 维度 | 评分思路 |
| --- | --- |
| 地域接受度 | 是否真的愿意去，是否能接受长期生活 |
| 专业匹配度 | 专业越窄、越精准，竞争可能越低 |
| 招录人数 | 人数越多，容错越高 |
| 限制条件 | 限制越多，符合者越少 |
| 服务年限 | 服务年限越长，越要慎重 |
| 加试成本 | 有专业科目或体测则成本更高 |
| 个人偏好 | 工作内容、平台、家庭距离、发展路径 |

建议最终保留三档：

- A 档：很想去、条件匹配、备考科目可控。
- B 档：能接受、条件匹配、竞争或限制一般。
- C 档：能报但不优先，主要作为备选。

## 报名前核对

报名公告出来后，再重新核对一次：

- 公告年份是否为当前考试年度
- 职位表是否为最新版本
- 专业目录是否更新
- 报考指南是否调整应届、基层经历、户籍口径
- 报名、缴费、准考证、笔试时间是否已记录
- 咨询电话是否保存

不要只凭上一年资料报名，上一年资料只能用来预筛和备考。
