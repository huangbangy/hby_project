## 职位表结构

2026 年全省计划招录公务员 10198 名，职位表分为 6 个工作表：

| 分表 | 招录人数 | 适合重点关注的人群 |
| --- | ---: | --- |
| 省直机关及直属单位 | 417 | 想优先考虑省直平台、专业匹配较强的人 |
| 市州及以下机关 | 6355 | 大多数普通岗位、县乡岗位、地域导向明显的人 |
| 法院系统 | 647 | 法学背景、法律职业资格相关人群 |
| 检察院系统 | 333 | 法学背景、法律职业资格相关人群 |
| 公安系统 | 669 | 能接受警察岗位、体测体检和年龄限制的人 |
| 综合行政执法队伍 | 1777 | 能接受行政执法卷、基层执法岗位的人 |

筛岗位时优先看这些字段：`考区`、`单位名称`、`单位层级`、`职位名称`、`笔试考试科目`、`招录人数`、`报考人员身份要求`、`基层工作年限要求`、`性别要求`、`最高年龄要求`、`最低学历要求`、`学位要求`、`专业要求`、`户籍要求`、`职业资格证书等要求`、`最低服务年限要求`、`笔试最低开考比例`、`体能测评项目和标准`。
