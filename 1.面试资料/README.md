# 面试资料索引

这个目录按面试准备流程整理，原始文件名保持不变。

## 01_简历与投递版本

存放简历源文件、Markdown 版本和不同岗位/公司的 PDF 投递版本。

- `01_PDF投递版本`：存放已经导出的 PDF 简历。
- `02_源文件与草稿`：存放 Markdown、草稿和其他非 PDF 简历文件。

## 02_岗位职责与JD

存放目标岗位职责、招聘 JD、岗位要求摘录，方便对照简历和准备回答。

## 03_面试记录

存放具体公司或具体岗位的面试记录、面试官问题、现场复盘。

## 04_题库与复盘

存放零散面试题、知识点复盘、简历亮点梳理、外部题目链接。

## 05_进展跟踪

存放投递、约面、面试结果和后续动作等进展记录。
